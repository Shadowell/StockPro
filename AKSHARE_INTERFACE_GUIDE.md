# AkShare 股票接口完整指南

## 概述
本文档总结了在股票分析项目中可用的AkShare接口，包含实时行情、历史数据、板块概念、资金流向等多种类型接口的详细信息。

## 一、A股实时行情接口

### 1. 东方财富A股实时行情
- **接口**: `stock_zh_a_spot_em()`
- **数据量**: 约5800+只股票
- **数据字段**: 
  - 序号、代码、名称、最新价、涨跌幅、涨跌额
  - 成交量、成交额、振幅、最高、最低、今开、昨收
  - 量比、换手率、市盈率-动态、市净率
  - 总市值、流通市值、涨速、5分钟涨跌、60日涨跌幅、年初至今涨跌幅
- **性能**: 稳定，推荐使用

### 2. 同花顺A股实时行情
- **接口**: `stock_zh_a_spot_ths()`
- **注意**: 当前版本可能存在稳定性问题

## 二、A股历史数据接口

### 1. A股历史行情
- **接口**: `stock_zh_a_hist(symbol, period='daily', start_date, end_date, adjust='')`
- **参数**:
  - `symbol`: 股票代码 (必填)
  - `period`: 周期 ('daily', 'weekly', 'monthly') 默认'daily'
  - `start_date`: 开始日期 (格式: 'YYYYMMDD')
  - `end_date`: 结束日期 (格式: 'YYYYMMDD')
  - `adjust`: 复权方式 ('', 'qfq', 'hfq') 空表示不复权
- **数据字段**: 日期、股票代码、开盘、收盘、最高、最低、成交量、成交额、振幅、涨跌幅、涨跌额、换手率
- **示例**: `stock_zh_a_hist('000001', period='daily', start_date='20240101', end_date='20240110')`

### 2. A股分时数据
- **接口**: `stock_zh_a_hist_min_em(symbol, period='1', adjust='0', start_date, end_date)`
- **参数**:
  - `symbol`: 股票代码
  - `period`: 周期 ('1', '5', '15', '30', '60')
  - `adjust`: 复权 ('0' 不复权, '1' 复权)
  - `start_date`: 开始时间 (格式: 'YYYY-MM-DD HH:MM:SS')
  - `end_date`: 结束时间 (格式: 'YYYY-MM-DD HH:MM:SS')

## 三、板块与概念接口

### 1. 东方财富概念板块
- **接口**: `stock_board_concept_name_em()`
- **数据字段**: 排名、板块名称、板块代码、最新价、涨跌额、涨跌幅、总市值、换手率、上涨家数、下跌家数、领涨股票、领涨股票-涨跌幅
- **数据量**: 约441个概念板块

### 2. 东方财富概念板块成分股
- **接口**: `stock_board_concept_cons_em(symbol)`
- **参数**: `symbol` - 板块代码
- **数据字段**: 序号、代码、名称、最新价、涨跌幅、涨跌额、成交量、成交额、振幅、最高、最低、今开、昨收、换手率、市盈率-动态、市净率

### 3. 东方财富行业板块
- **接口**: `stock_board_industry_name_em()`

### 4. 东方财富行业板块成分股
- **接口**: `stock_board_industry_cons_em(symbol)`

### 5. 同花顺概念板块
- **接口**: `stock_board_concept_name_ths()`
- **数据字段**: name、code
- **数据量**: 约375个概念板块

## 四、热门股票与排行榜

### 1. 东方财富热门股票
- **接口**: `stock_hot_rank_em()`
- **数据字段**: 当前排名、代码、股票名称、最新价、涨跌额、涨跌幅
- **数据量**: 100只热门股票

### 2. 问财热门股票
- **接口**: `stock_hot_rank_wc()` (当前版本可能不可用)

### 3. 涨停池
- **接口**: `stock_zt_pool_em(date)`
- **参数**: `date` - 日期 (格式: 'YYYYMMDD' 或 'YYYY-MM-DD')
- **数据字段**: 序号、代码、名称、涨跌幅、最新价、成交额、流通市值、总市值、换手率、封板资金、首次封板时间、最后封板时间、炸板次数、涨停统计、连板数、所属行业
- **数据量**: 当日涨停股票

### 4. 其他涨停相关
- `stock_zt_pool_previous_em()` - 昨日涨停池
- `stock_zt_pool_strong_em()` - 强势股池
- `stock_zt_pool_sub_new_em()` - 次新股池

## 五、资金流向接口

### 1. 个股资金流
- **接口**: `stock_individual_fund_flow(symbol)`
- **参数**: `symbol` - 股票代码

### 2. 市场资金流
- **接口**: `stock_market_fund_flow()`
- **数据字段**: 日期、上证收盘价、上证涨跌幅、深证收盘价、深证涨跌幅、主力净流入净额、主力净流入净占比、超大单净流入净额、超大单净流入净占比、大单净流入净额、大单净流入净占比、中单净流入净额、中单净流入净占比、小单净流入净额、小单净流入净占比

### 3. 行业资金流排名
- **接口**: `stock_sector_fund_flow_rank()`

### 4. 个股资金流排名
- **接口**: `stock_individual_fund_flow_rank()`

## 八、在项目中的实际应用

根据项目API文档，以下接口在项目中被实际使用：

| 功能 | 接口 | 用途 |
|------|------|------|
| 股票筛选 | `stock_zh_a_spot_em` + `stock_zh_a_hist` | 获取股票实时行情和历史数据 |
| 热门概念 | `stock_board_concept_name_em` | 获取概念板块数据 |
| 概念资金流 | `stock_fund_flow_concept(symbol="即时")` | 获取概念板块资金流 |

| 涨停池 | `stock_zt_pool_em` | 获取涨停股票数据 |
| 概念分时 | `stock_board_concept_hist_min_em` | 获取概念板块分时数据 |
| 成份股 | `stock_board_concept_cons_em` | 获取概念板块成分股 |
| 历史数据 | `stock_zh_a_hist` | 获取股票历史数据 |
| 分时数据 | `stock_zh_a_minute` | 获取股票分时数据 |

## 九、使用建议与注意事项

### 1. 接口优先级策略
- 优先使用同花顺接口，若不可用则降级至东方财富接口
- 实施重试机制和备选方案

### 2. 错误处理
- 所有接口都应做好异常处理
- 网络接口可能不稳定，需要容错机制

### 3. 缓存策略
- 对于频繁访问的数据，使用适当的缓存机制
- 设置合理的缓存过期时间

### 4. 请求频率
- 避免过于频繁地调用接口
- 遵循API使用规范，防止被限制访问

### 5. 数据字段差异
- 不同接口返回的列名可能不同
  - 东方财富概念板块: '板块代码', '板块名称'
  - 同花顺概念板块: 'code', 'name'
  - 热门股票排行: '当前排名', '股票名称'
- 使用前先检查返回数据的列名

### 6. 版本兼容性
- AkShare接口可能会随版本变化
- 定期验证接口可用性

## 十、常见问题排查

### 1. 接口不可用
- 检查AkShare版本是否最新
- 验证接口名称是否正确
- 检查参数格式是否符合要求

### 2. 数据为空
- 确认参数正确性
- 检查日期是否为交易日
- 验证股票代码是否存在

### 3. 网络错误
- 检查网络连接
- 降低请求频率
- 实施重试机制

# Stock Analysis App - Frontend

## Development

```bash
npm install
npm run dev
```

## Type Check

```bash
npm run check
```

## Pages

- `/`：Strategy Filter
- `/market`：Market Overview
- `/analysis`：Data Processing & Analysis
- `/ai`：AI Analysis
- `/news-calendar`：News & Calendar

后端 API 默认前缀为 `/api/v1`（详见根目录 `README.md` 与 `docs/api.md`）。

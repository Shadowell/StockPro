# StockApp 产品安装与使用指南

## 1. 产品简介
StockApp 是一款专为职业投资者设计的量化分析与市场监控桌面应用。它集成了实时行情监控、热门概念追踪、AI 智能股票分析以及自动化数据同步功能。

## 2. 系统要求
- **操作系统**：macOS 12+, Windows 10/11, Ubuntu 22.04+
- **环境依赖**：
    - Python 3.10+
    - Node.js 18+
    - SQLite 3

## 3. 安装步骤

### 3.1 获取源代码
```bash
git clone <repository_url>
cd stock_app
```

### 3.2 后端配置
1. 创建并激活虚拟环境：
```bash
python3 -m venv venv
source venv/bin/activate  # Windows 使用 venv\Scripts\activate
```
2. 安装 Python 依赖：
```bash
pip install -r backend/requirements.txt
```
3. 配置环境变量：
   在 `backend/.env` 中配置相关参数（如 `OPENAI_API_KEY` 等）。

### 3.3 前端配置
1. 进入前端目录并安装依赖：
```bash
cd frontend
npm install
```

## 4. 启动应用

### 4.1 开发模式启动
1. 启动后端服务：
```bash
cd backend
python main.py
```
2. 启动前端预览：
```bash
cd frontend
npm run dev
```

### 4.2 首次数据同步
运行以下脚本以拉取最近半年的全量股票历史数据：
```bash
python3 backend/initial_sync.py
```

## 5. 功能使用说明

### 5.1 首页 - 策略看板
- **市场洞察**：顶部的卡片展示当前市场最热板块、情绪指数及异动告警。
- **高潜力股**：查看系统根据内置策略筛选出的个股，点击任意行可展开详细 K 线。

### 5.2 市场概览
- **热门概念**：按涨幅排序的板块列表，支持查看其资金流向。
- **概念拖拽**：您可以根据关注度，自由拖拽板块详情页的标签（如：龙头股、分时K线）进行排序，系统将自动记住您的习惯。

### 5.3 AI 研报分析
- 在搜索框输入股票代码或名称，选择后点击“开始分析”。
- 系统将结合最新行情、基本面及技术指标，生成专业级的交易计划与趋势判断。

## 6. 数据同步说明
- **定时同步**：系统每日 18:00 会自动拉取全市场数据并更新本地数据库。
- **离线访问**：即使在无网络环境下，您仍可查看已同步过历史数据的个股日线图。

## 7. 常见问题 (FAQ)
- **Q: 为什么 K 线显示“暂无数据”？**
- A: 请确保已运行过 `initial_sync.py` 或后端服务正在运行且能正常访问网络。
- **Q: 颜色配置可以修改吗？**
- A: 系统目前默认采用 A 股传统的“红涨绿跌”配色，符合国内投资者习惯。
